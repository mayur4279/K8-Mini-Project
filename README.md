# k8-mini-project 

1. clone this repository.
   ```bash
   git clone https://github.com/mayur4279/k8-mini-project.git
   ```

2. Run all this files using follwing command.
   ```bash
   kubectl  apply -f  .
   ```

3. Get The node IP using follwing command.
   ```bash
   kubectl  get nodes  -o wide
   ```
